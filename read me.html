About DX Pong!

DX Pong is yet another Pong game, but I hope that this one is a bit more special that the hundreds of others you find on the internet because :

  • It has clever AI - the computer does not blindly follow the ball. Rather it predicts it's path and can even account for the ball rebounding off the sides.

  • It is difficult, but not impossible to win

  • It has cool graphics

  • It has cool sound effects

  • It has decent ball physics - for example you can put spin on the ball with the bat.

  • Also there's the fact that this version has taken just one day to make.

PC Requirements

  • You must have DirectX 7 or higher installed on your machine.

  • You must have a monitor which supports 640 x 480 resolution graphics and 16 bit colors.

  • The game runs at over 50 frames per second on my 400Mhz PC, but I'm sure it'll run fine on slower machines too.

How to Play

  • Just click on the .exe and use the arrow keys to move up and down. You control the bat on the left side of the screen. There's not much to it! But the best games are always the simplest.

  • If you run the game from Visual Basic, the code will also print the frame rate to the debug window every second, so you can really see how fast the game is!

  • You can also Click Here to start the game.
